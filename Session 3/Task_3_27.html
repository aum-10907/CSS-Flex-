A **favicon** (short for *favorite icon*) is a small icon that represents a website. It appears in the browser tab, bookmarks, browser history, and sometimes in search results, making it easier for users to identify a website. Favicons are typically 16×16, 32×32, or 48×48 pixels in size and are commonly saved in `.ico`, `.png`, or `.svg` formats. They are added to a webpage using the `` tag inside the `` section of an HTML document. Using a favicon improves a website's branding, gives it a more professional appearance, and helps users quickly recognize the site when multiple tabs are open.
